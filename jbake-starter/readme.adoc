JBake is a Java based, open source, static site/blog generator for developers & designers

https://jbake.org/

== Getting started

  mkdir hello-site
  ../jbake-2.6.4-bin/bin/jbake -i
  ../jbake-2.6.4-bin/bin/jbake -s
  open http://localhost:8820
